{\bf Dataset.} We evaluate several models on a synthetic dataset generated from a linear dynamical system. The transition matrix  is a  dimension Wishart random matrix normalized to have operator norm equals . The noise  in the state transition follows from a -dimensional Gaussian distribution. The input at each time-step  follows from a -dimensional standard Gaussian distribution, which is observable to the model. Finally, We add seasonality of  different periodicity by adding cosine signal as the input to the linear dynamical system, but hidden from the prediction model. We generate  different time series which shares the same model parameter, input  and seasonality input, with the only difference coming from the randomness in the state transition. 
We set look-back window to be length , and horizon also to length . For each time-series, we use the first  steps for training, next  steps for validation, and the final  steps for testing, which results in  examples for training,  examples for validation, and  examples for testing. 

{\bf Baselines and Setup.}
We evaluate three models on our synthetic dataset: linear, long short-term memory (LSTM) and Transformer. Our linear model is a direct linear map between the history in look-back window and future. We use a one layer LSTM with dimension 128. For Transformer, we use a two layer self-attention layers with dimension 128, combined with a one hidden layer feed-forward network with 128 hidden units.

{\bf Results.} We present Mean Squared Error (MSE) for all models in Table~\ref{tab:linear_result}. For all models, we report the mean and standard deviation out of 3 independent runs for each setting. We also plot the actuals (ground truth) vs the predictions from Linear, LSTM and Transformer models. We also load time derived features as covariates.
